# 进销存 ERP 系统

基于飞书多维表格 + lark-cli 构建的轻量进销存管理系统。适合小团队管理货品台账、入库出库流水、库存预警和供应商信息。

## 快速开始

### 1. 确保已授权

```bash
lark-cli auth login --domain base,im,docs
```

### 2. 一句话初始化

直接用自然语言告诉 AI：

```
帮我创建一个进销存系统
```

AI 会自动完成以下步骤：
- 创建多维表格 Base
- 创建 4 张数据表（货品台账、入库记录、出库记录、供应商）
- 添加所有字段（物品名称、规格、类别、单位、最低库存等）
- 配置双向关联（台账↔入库、台账↔出库、台账↔供应商）
- 创建 Lookup 和 Formula 计算字段（累计入库、累计出库、剩余库存、库存状态）

初始化大约需要 2-3 分钟（因限流保护，每个字段间隔 1.5 秒）。

### 3. 开始使用

```
新增货品：米色T恤，规格 S/M/L，最低库存 20
```

## 定时自动化

可以在 Claude Code 中配置定时任务自动执行库存相关操作：

### 每日低库存预警（工作日 9:00）

```
CronCreate: cron="0 9 * * 1-5", prompt="检查所有货品的库存状态，将低库存货品发送预警通知到仓库管理群", recurring=true
```

### 每周库存报表（每周五 17:00）

```
CronCreate: cron="0 17 * * 5", prompt="生成本周的入库出库汇总报表，保存为飞书文档", recurring=true
```

## 与其他 Skill 的关系

- **数据层**：依赖 `lark-base`（多维表格操作）
- **通知层**：依赖 `lark-im`（飞书消息发送）
- **报表层**：依赖 `lark-doc`（飞书文档生成）
- **独立使用**：不依赖其他工作流 Skill，可单独使用
